Before using mods:

- close the game before installing or uninstalling

- remove the mod if the game shows XML errors
- follow the server owner's rules when joining multiplayer
- use the Easy Anti-Cheat setting required by your server or play mode

## What The Package Does Not Include

The game modlet is XML-only. The full Windows package also includes readable `.bat` and `.ps1` helper scripts for installing, uninstalling, scanning, tuning, and optional Brutal Science `serverconfig.xml` backup/restore.

The game modlet does not include:

- DLL files
- EXE files
- Harmony patches
- custom scripts
- game assets copied from 7 Days to Die

The included `.bat` and `.ps1` files are installer helper scripts. They copy or remove the mod folder; they do not patch the game executable.
