# Versioned Solution Lanes

This directory contains version-specific solution references.

- `4.0.1/` is the preserved historical source/package copy.
- `4.1.0/` is the active 4.1.x promotion lane.

The working 4.1.0 source remains in the parent solution folder until the
release tooling is updated to use this version boundary. Do not copy the
parent solution into a child of itself; that creates recursive version trees.
